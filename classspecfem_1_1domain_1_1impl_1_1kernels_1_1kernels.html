Detailed Description

template<class medium, class qp_type>
class specfem::domain::impl::kernels::kernels< medium, qp_type >

Kernels object used to compute elemental kernels.

This object consists of various Kokkos kernels used to compute elemental contributions from different types of element, sources, and receivers.

The template parameters are inherited from the domain class.

Template Parameters
mediumclass defining the domain medium. Separate implementations exist for elastic, acoustic or poroelastic media
qp_typeclass used to define the quadrature points either at compile time or run time
