Detailed Description

template<typename base_element>
struct specfem::domain::impl::elements::container< base_element >

Struct to hold a pointer to an element.

Kokkos views have a syntantic problem when creating a view of pointers. For example, let's say we have want to create a view of pointers to int. The syntax would be: View<int* *>, this is ambigous since the compiler doesn't know if the * is part of 2-dimensional view type or a pointer to int type. To avoid this problem, we create a struct which holds a pointer to an element. This struct is then used to create a view of structs. This allows us to create a view of pointers to elements.
