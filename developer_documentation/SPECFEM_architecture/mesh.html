Mesh developer guideÔÉÅ

mesh struct defines IO routines and data structures to store data related to spectral element mesh. The mesh itself is generated using either the internal mesher or an external mesher. The current implementation only supports meshes generated using MESHFEM2D. This guide describes the data structures and IO routines required to incorporate a new mesher.
