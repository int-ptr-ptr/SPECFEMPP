Detailed Description

Elastic domain class.

Elastic domains implementation details:

  • field -> Displacements along the 2 dimensions (idim) for every global point (iglob) stored as a 2D View field(iglob, idim)
  • field_dot -> Velocity along the 2 dimensions (idim) for every global point (iglob) stored as a 2D View field(iglob, idim)
  • field_dot_dot -> Acceleration along the 2 dimensions (idim) for every global point (iglob) stored as a 2D View field(iglob, idim)
