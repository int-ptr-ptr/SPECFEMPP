template<typename quadrature_points>
class specfem::domain::impl::elements::element< specfem::enums::element::dimension::dim2, specfem::enums::element::medium::acoustic, quadrature_points >

Acoustic element in 2D.

Base class to define specialized acoustic elements in 2D. This class defines pure virtual methods to compute the gradient and stress at the quadrature points of the element. The specific implementaions are left to the derived classes, e.g.

Compute the mass matrix component ($ m_{\alpha, \beta} $) for a given quadrature point.

Mass matrix is given by \f$ M = \sum_{\Omega_e} \sum_{\alpha, \beta} \omega_{\alpha} \omega_{\beta} m_{\alpha, \beta} \f$
