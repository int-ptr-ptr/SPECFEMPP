Detailed Description

template<typename qp_type>
class specfem::domain::impl::elements::element< specfem::enums::element::dimension::dim2, specfem::enums::element::medium::elastic, qp_type >

Elastic element in 2D.

Base class to define specialized elastic elements in 2D. This class defines pure virtual methods to compute the gradient and stress at the quadrature points of the element. The specific implementaions are left to the derived classes.

Template Parameters
quadrature_pointsType for number of quadrature points defined either at compile time or run time

Compute the mass matrix component ($ m_{\alpha, \beta} $) for a given quadrature point.

Mass matrix is given by \f$ M = \sum_{\Omega_e} \sum_{\alpha, \beta} \omega_{\alpha} \omega_{\beta} m_{\alpha, \beta} \f$
