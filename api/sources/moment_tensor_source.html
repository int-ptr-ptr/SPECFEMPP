Locate source within the mesh.

Given the global cartesian coordinates of a source, locate the spectral element and xi, gamma value of the source

Parameters:

  • h_ibool – Global number for every quadrature point

  • coord – (x, z) for every distinct control node

  • xigll – Quadrature points in x-dimension

  • zigll – Quadrature points in z-dimension

  • nproc – Number of processors in the simulation

  • coorg – Value of every spectral element control nodes

  • knods – Global control element number for every control node

  • npgeo – Total number of distinct control nodes

  • ispec_type – material type for every spectral element

  • mpi – Pointer to specfem MPI object
