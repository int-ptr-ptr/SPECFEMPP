Receiver API

The receiver class provides methods to read, store and process seismogram station data.

void locate(const specfem::kokkos::HostView2d<type_real> coord, const specfem::kokkos::HostMirror3d<int> h_ibool, const specfem::kokkos::HostMirror1d<type_real> xigll, const specfem::kokkos::HostMirror1d<type_real> zigll, const int nproc, const specfem::kokkos::HostView2d<type_real> coorg, const specfem::kokkos::HostView2d<int> knods, const int npgeo, const specfem::kokkos::HostMirror1d<specfem::enums::element::type> ispec_type, const specfem::MPI::MPI *mpi)

Locate station within the mesh.

Given the global cartesian coordinates of a station, locate the spectral element and xi, gamma value of the station

Parameters:

  • h_ibool – Global number for every quadrature point

  • coord – (x, z) for every distinct control node

  • xigll – Quadrature points in x-dimension

  • zigll – Quadrature points in z-dimension

  • nproc – Number of processors in the simulation

  • coorg – Value of every spectral element control nodes

  • knods – Global control element number for every control node

  • npgeo – Total number of distinct control nodes

  • ispec_type – material type for every spectral element

  • mpi – Pointer to specfem MPI object
