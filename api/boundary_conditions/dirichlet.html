Dirichlet boundary conditions

Definition

template<typename dim, typename medium, typename property, typename qp_type>
class dirichlet

Dirichlet boundary condition.

Template Parameters:

  • dim – Dimension of the boundary.

  • medium – Medium type for the element where the boundary is located.

  • property – Property type for the element where the boundary is located.

  • qp_type – Quadrature points object to define the quadrature points either at compile time or run time.

Interface

Parameters

  • dimension:

    The dimension of the element.

    • dim2(): A two dimensional element.

    • dim3(): A three dimensional element.

  • medium:

    The medium of the element.

    • elastic(): An elastic element.

    • acoustic(): An acoustic element.

  • quadrature_points_type:

    The quadrature points of the element.

    • static_quadrature_points< NGLL >(): A static quadrature point set.

  • properties:

    The properties of the element. The properties describe any specializations made the implementation.

    • Type of element:

      • isotropic(): An isotropic element.

Template Implementation

  • Dirichlet implementation of various elements

    • dirichlet< typename dimension , typename medium , typename property , typename quadrature_points >()
