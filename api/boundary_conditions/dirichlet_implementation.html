template<typename dim, typename medium, typename property, typename qp_type>
class dirichlet

Dirichlet boundary condition.

Template Parameters:

  • dim – Dimension of the boundary.

  • medium – Medium type for the element where the boundary is located.

  • property – Property type for the element where the boundary is located.

  • qp_type – Quadrature points object to define the quadrature points either at compile time or run time.
