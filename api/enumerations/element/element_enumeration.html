Element

Enumerations that define a type of element.

Dimension

The dimension of the element.

  • dim2(): A two dimensional element.

  • dim3(): A three dimensional element.

Medium

The medium of the element.

  • elastic(): An elastic element.

  • acoustic(): An acoustic element.

Medium enumerations to tag the element with a medium.

enum class specfem::enums::element::type

type of element

This is primarily used to label the element as elastic, acoustic or poroelastic.

Values:

enumerator elastic

elastic element

enumerator acoustic

acoustic element

enumerator poroelastic

poroelastic element

Quadrature Points

The quadrature points of the element.

  • static_quadrature_points< NGLL >(): A static quadrature point set.

Elemental Properties

The properties of the element. The properties describe any specializations made the implementation.

  • Type of element:

    • isotropic(): An isotropic element.

Property enumerations to tag the element with a property.

enum class specfem::enums::element::property_tag

Values:

enumerator isotropic

isotropic material

Boundary Conditions

Defines the boundary conditions on the element. The boundary conditions are used to modify the element’s force contribution to the global force vector.

  • boundary_conditions: The boundary conditions of the element.

Boundary condition enumerations to tag the element with a boundary condition.

enum class specfem::enums::element::boundary_tag

Values:

enumerator none

no boundary

enumerator acoustic_free_surface

free surface boundary for acoustic elements

enumerator stacey

stacey boundary for elements

enumerator composite_stacey_dirichlet

composite boundary for acoustic elements
