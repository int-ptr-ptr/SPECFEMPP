void zwgljd(specfem::kokkos::HostMirror1d<type_real> z, specfem::kokkos::HostMirror1d<type_real> w, const int np, const type_real alpha, const type_real beta)

Generate np Gauss-Lobatto-Jacobi points and the weights associated with Jacobi polynomials of degree n = np-1.

Note

alpha and beta coefficients must be greater than -1. Legendre polynomials are special case of Jacobi polynomials just by setting alpha and beta to 0.

Parameters:

  • z – HostMirror1d where GLL points will be stored

  • w – HostMirror1d where GLL weights will be stored

  • np – Number of GLL points

  • alpha – Alpha value of the Jacobi polynomial

  • beta – Beta value of the Jacobi polynomial

void zwgjd(specfem::kokkos::HostMirror1d<type_real> z, specfem::kokkos::HostMirror1d<type_real> w, const int np, const int alpha, const int beta)

Compute the GLL points and weights of jacobi polynomials inside the open interval (-1,1)

Note

This function uses gll_utils::jacg and gll_utils::jacw for calculate the GLL points and weights

Warning

The weights are not normalized by the this function w(i) = w(i)*(1-z(i)**2)

Parameters:

  • z – HostArray where GLL points will be stored

  • w – HostArray where GLL weights will be stored

  • np – Degree of the Jacobi polynomial

  • alpha – Alpha value of the Jacobi polynomial

  • beta – Beta value of the Jacobi polynomial
