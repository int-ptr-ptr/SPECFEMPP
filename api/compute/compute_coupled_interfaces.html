Coupled Interfaces Struct

The coupled_interfaces struct is used to read and store coupling information for every coupled interface within the domain.

struct coupled_interfaces

Struct used to store all interface information at every interfacial quadrature point within the mesh.

Public Functions

Construct a new coupled interfaces object.

Parameters:

  • h_ibool – Index for every quadrature point in the mesh

  • coord – Coordinates of every quadrature point in the mesh

  • h_ispec_type – Element type (acoustic/elastic/poroelastic) for every element in the mesh

  • coupled_interfaces – Interface information for every interface read from mesher
