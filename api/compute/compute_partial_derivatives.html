Partial Derivatives Struct

The partial_derivatives struct is used to store partial derivatives for all GLL points in the mesh.

struct partial_derivatives

Partial derivates matrices required to compute integrals.

The matrices are stored in (ispec, iz, ix) format

Public Functions

inline partial_derivatives()

Default constructor.

partial_derivatives(const int nspec, const int ngllz, const int ngllx)

Constructor to allocate views.

Parameters:

  • nspec – Number of spectral elements

  • ngllz – Number of quadrature points in z direction

  • ngllx – Number of quadrature points in x direction
