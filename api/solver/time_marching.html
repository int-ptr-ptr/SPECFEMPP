Time Marching Solver

Time marching explicit solver

template<typename qp_type>
class time_marching : public specfem::solver::solver

Time marching solver class.

Implements a forward time marching scheme given a time scheme and domains. Currently only acoustic and elastic domains are supported.

Template Parameters:

qp_type – Type defining number of quadrature points either at compile time or run time
