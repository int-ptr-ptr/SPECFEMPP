template<class medium, class qp_type>
class kernels

Kernels object used to compute elemental kernels.

This object consists of various Kokkos kernels used to compute elemental contributions from different types of element, sources, and receivers.

The template parameters are inherited from the domain class.

Template Parameters:

  • medium – class defining the domain medium. Separate implementations exist for elastic, acoustic or poroelastic media

  • qp_type – class used to define the quadrature points either at compile time or run time

Construct a new kernels object.

Parameters:

  • ibool – Global index for every GLL point in the SPECFEM simulation

  • partial_derivatives – struct used to store partial derivatives at every GLL point

  • properties – struct used to store material properties at every GLL point

  • sources – struct used to store information about sources

  • receives – struct used to store information about receivers

  • quadx – Pointer to quadrature points in X direction

  • quadz – Pointer to quadrature points in Z direction

  • quadrature_points – quadrature points object to define number of quadrature points either at compile time or run time

  • field – wavefield inside the domain

  • field_dot – derivative of wavefield inside the domain

  • field_dot_dot – double derivative of wavefield inside the domain

  • mass_matrix – mass matrix for every GLL point inside the domain
