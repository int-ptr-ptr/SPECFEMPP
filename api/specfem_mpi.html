SPECFEM abstractions for MPI interface

Always manage MPI communication using a global instantiation of MPI class. Never call MPI routines directly.

class MPI
#include <specfem_mpi.h>

MPI class instance to manage MPI communication.

Note

If specfem is compiled without MPI then world_size = 1 and my_rank = 0 Additionally, many routines are just empty to optimize performance.
