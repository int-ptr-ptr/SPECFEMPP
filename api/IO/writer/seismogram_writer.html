Seismogram Writer

The seismogram provides methods to write seismograms to a file.

class seismogram : public specfem::writer::writer

Seismogram writer class to write seismogram to a file.

Public Functions

inline seismogram(std::vector<std::shared_ptr<specfem::receivers::receiver>> &receivers, specfem::compute::receivers &compute_receivers, const specfem::enums::seismogram::format type, const std::string output_folder, const type_real dt, const type_real t0, const int nstep_between_samples)

Construct a new seismogram writer object.

Parameters:

  • receivers – Vector of pointers to specfem::receivers::receiver objects

  • compute_receivers – Pointer to specfem::compute::receivers object

  • type – Format of the output file

  • output_folder – path to output folder where results will be stored

  • dt – Time interval between subsequent timesteps

  • t0 – Solver start time

  • nstep_between_samples – number of timesteps between seismogram sampling (seismogram sampling frequency)
