Mesh Material Information

The material_ind struct is used to store material information for every spectral element in the mesh.

struct material_ind

Public Functions

inline material_ind()

Default constructor.

material_ind(const int nspec, const int ngnod)

Constructor used to allocate views.

Parameters:

  • nspec – Number of spectral elements

  • ngnod – Number of control nodes per spectral element

material_ind(std::ifstream &stream, const int ngnod, const int nspec, const int numat, const specfem::MPI::MPI *mpi)

Constructor used to allocate and assign views from fortran database file.

Parameters:

  • stream – Stream object for fortran binary file buffered to material definition section

  • ngnod – Number of control nodes per spectral element

  • nspec – Number of spectral elements

  • numat – Total number of different materials

  • mpi – Pointer to a MPI object

Public Members

specfem::kokkos::HostView1d<int> region_CPML

Defines the type of PML. 

if (region_CPML(ispec) == 0) means ispec is not a PML element
if (region_CPML(ispec) == 1) means ispec is a X PML element
if (region_CPML(ispec) == 2) means ispec is a Z PML element
if (region_CPML(ispec) == 3) means ispec is a XZ PML element
specfem::kokkos::HostView1d<int> kmato

Defines material specification number

specfem::kokkos::HostView2d<int> knods

Defines global control element number for every control node. 

for ispec : nspec
   for ia : ngnod
       // ipgeo defines global element control number
       ipgeo = knods(ia, ispec)
