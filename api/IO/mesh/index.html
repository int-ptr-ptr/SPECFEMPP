Mesh Input Module

The mesh module provides set of structs and functions to read and store mesh database files generated by MESHFEM2D. The mesh database files need to be stored in fortran binary format.

Mesh Struct API

struct mesh

Mesh Interface.

The mesh is implemented as a C++ struct. The mesh struct defines all the variables nacessary to populate structs within specfem::compute namespace
