Detailed Description

template<typename quadrature_points>
class specfem::domain::impl::sources::source< specfem::enums::element::dimension::dim2, specfem::enums::element::medium::acoustic, quadrature_points >

Elemental source class for 2D acoustic media.

Base class for all 2D acoustic elemental sources. This class contains pure virtual methods that must be implemented by the template specializations.

Template Parameters
quadrature_pointsNumber of Gauss-Lobatto-Legendre quadrature points defined at compile time or at runtime
